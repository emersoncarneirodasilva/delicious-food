# Delicious Food

Este é um projeto de um site de restaurante com design moderno, dinâmico e de interface bonita.

Foram utilizadas as tecnologias: Next.js, Tailwind CSS, TypeScript, React Icons.

Construído a partir do vídeo (https://bit.ly/3Pm1zXE) do canal do YouTube "The Brave Coders" (https://bit.ly/46bBBgt).
